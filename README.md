## **Overview üîç**

The Technoskill 1.0 Competition is a competition held by the PIPTEK division of IME FTUI 2024. This competition is in the field of ***Fullstack Web Development*** that involves the technical side of the frontend and backend in the form of Website creation.

The competition is held on July 26 - August 5, 2024, after the series of workshops have been completed. This competition aims to channel students' ideas and creativity in learning how the website creation flow can work and can hone students' abilities, especially in the field of software, especially web development.

### ***This project provides a comprehensive solution for managing employees in a company, designed to streamline HR tasks with a user-friendly interface.***


## **Features üíª**

- ### Login Page:

This is the first page the user will see when they launch the app on their computer. In this page, the user will be asked to log in to the site using their account before accessing the database.

<p align="center">
<img src="FIGMA\LoginPage.jpeg" width="500" style="border-radius: 10px;">
</p>

If the user tries to login without entering a username and/or password an error will show up. Similarly, if the user tries to login using the wrong username and/or password for an account, a different error will show up *(These errors wild be shown in practice in the youtube demo in the end of the page)*. If the user hasn't made an account yet, they can click on the register tab in the navbar or click the following text below to enter the register page.

<p align="center">
<img src="FIGMA\LoginPagetoRegister.jpeg" width="250" style="border-radius: 10px;">
</p>

- ### Register Page:

In the register, the user will be able to make a new account to access the database. In order to do that, first the user will have to enter a username, email, and password for the account. The user will also have to retype the password to ensure security. If both passwords doesn't match or if the email is invalid, the user will get an error message like in the login page.

<p align="center">
<img src="FIGMA\RegisterPage.jpeg" width="500" style="border-radius: 10px;">
</p>

After successfully registering an account, the user will be directed to the login page again to log into the account that they have made.

- ### Home Page:

After logging in to an account, the user will be directed to the home page of the app. In here, the user will be able to view the database by clicking the button below: 

<p align="center">
<img src="FIGMA\HomePage.jpeg" width="500" style="border-radius: 10px;">
</p>

If the user haven't added any employees, then the database will be shown like this:

<p align="center">
<img src="FIGMA\Database.jpeg" width="500" style="border-radius: 10px;">
</p>

If the user has added some employees, then the database will look something like this:

<p align="center">
<img src="FIGMA\HomePageDatabase.jpeg" width="500" style="border-radius: 10px;">
</p>

To view an employees details from the home page, the user can click on the employee's name that they want to view the details of like this:

<p align="center">
<img src="FIGMA\HomePageEdit.jpeg" width="500" style="border-radius: 10px;">
</p>

In this page, the user will also be able to edit any details that they want and save the changes thet they have made. They can also delete the employee by clicking the buttons below:

<p align="center">
<img src="FIGMA\HomePageEditButtons.jpeg" width="250" style="border-radius: 10px;">
</p>

- ### My Info Page:

The my info page can be accessed from the home page using the navbar on top shown below:

<p align="center">
<img src="FIGMA\HomePagetoMyInfo.jpeg" width="250" style="border-radius: 10px;">
</p>

In the my info page, the user will be able to view and change their account details as shown below:

<p align="center">
<img src="FIGMA\MyInfoPage.jpeg" width="500" style="border-radius: 10px;">
</p>

On the left window, the user could see their current name and email. They can also see their profile picture, gender, date of birth, and address if they have them or add them manually if they haven't already. In the right window, the user can change their username, email, and password.

- ### Add New Page:

In the add new page, the user will be able to add a new employee to the database by inputting it's name, division, and salary as shown below:

<p align="center">
<img src="FIGMA\AddNewPage.jpeg" width="500" style="border-radius: 10px;">
</p>

After clicking the button to add a new employee, the user will be sent back to the home page and can view the employee they just added in the database.
